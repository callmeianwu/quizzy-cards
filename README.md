## ✨ Features

- 📝 **Create Custom Flashcard Sets** - Make personalized study materials
- 🔄 **Import/Export Functionality** - Share sets with friends or between devices
- 🤖 **AI-Generated Flashcards** - Use GPT-4 to create study cards in seconds
- 📊 **Learning Algorithm** - Smart spaced repetition system to optimize your study time
- 📱 **Mobile Friendly** - Responsive design works on all your devices

## 🚀 Getting Started

### Local Installation

1. **Download** the application files
2. **Unzip** to your preferred location
3. **Open** the `index.html` file in your web browser

No server setup or installation required!

## 💡 How to Use

### Creating Flashcard Sets

1. Enter a title for your set
2. Add questions and answers in the format:
   ```
   Q: What is the capital of France?
   A: Paris
   ```
3. Click "Create Set" to save

### Studying Flashcards

1. Click "Study" on any flashcard set
2. Click on the card to flip between question and answer
3. Rate your knowledge of the card:
   - **Easy** - You know it well
   - **Medium** - You somewhat remember
   - **Hard** - You need more review

### AI-Generated Flashcards

1. Click the robot icon in the bottom left
2. Enter your OpenAI API key
3. Provide a title and describe what you want to study
4. Select the number of cards you want
5. Click "Generate My Flashcards"


## 💾 Data Storage

All your flashcard data is stored locally in your browser. Nothing is sent to any server unless you're using the AI feature, which requires an OpenAI API key.

## 🔄 Import/Export

- **Export** your sets to share with friends
- **Import** sets others have shared with you
- Files use a simple JSON format for maximum compatibility

## 🔧 Advanced Features

- **Spaced Repetition** - Cards you find difficult appear more frequently
- **Mastery Tracking** - Monitor your progress as you learn
- **Custom Formatting** - Supports basic formatting in your flashcards
